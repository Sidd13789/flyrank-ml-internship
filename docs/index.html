Abstract

This project analyzes the FlyRank ML Internship dataset to study content performance using machine learning techniques. A Random Forest classifier was developed and evaluated against the Week-4 baseline using the same evaluation procedure. The model achieved strong predictive performance on the evaluation dataset. Additional validation and leakage checks were performed to improve the trustworthiness of the results. The findings should be interpreted as decision-support rather than automated business decisions.

1. Research Question

Can machine learning identify content that may require refresh or optimization using historical search and engagement signals?

2. Dataset

  • Dataset: FlyRank ML Internship Dataset
  • Approximately 30,000 anonymized content records
  • No client names or private information included
  • Target Variable: trend_direction
  • Public-safe dataset used for educational purposes

3. Methodology

  • Baseline model developed during Week-4
  • Random Forest Classifier selected for modeling
  • 80/20 Train-Test Split
  • Categorical variables encoded using LabelEncoder
  • Model evaluated using Accuracy, Precision, Recall and F1 Score
  • Leakage audit performed before interpreting results

4. Results (Model vs Baseline)

Model Metric
Week-4 Baseline Your Week-4 Baseline Score
Random Forest 99.95%

The Random Forest model achieved higher predictive performance than the baseline using the same evaluation procedure. These observations are intended for decision-support only.

5. Limitations

  • The model was evaluated only on the internship dataset.
  • Results may not generalize to unseen production data.
  • The work does not replace expert human review.
  • Further validation with grouped or time-aware splits is recommended.

6. Ranked Recommendations

  1. Review pages with declining trend direction.
  2. Prioritize high-impression content for refresh.
  3. Use model predictions only as decision-support.
  4. Perform manual review before publishing updates.
  5. Monitor model performance over time.
